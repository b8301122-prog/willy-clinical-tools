# 投影片設計一致性原則 (Law of Similarity)

> **書名**:你的簡報就是比 AI 強:47 個人類才懂的 PPT 技巧
> **章節**:3-9 如何讓簡報設計具有一致性?

---

## 核心概念
人會下意識地把視覺上具有相同特徵的要素解讀成同一族群,這在心理學稱為「相似法則」(Law of Similarity)。運用此法則可大幅提高簡報的易讀性與專業感。

不需所有條件都相同,只要**至少具備一個共同特徵**(顏色、形狀、大小等),觀眾就會自動將其辨識為同族群。

## 四大一致性檢查重點

### 1. 投影片設計要統一
- 整份簡報使用統一的版型 / template
- 每頁配置一致的元素(如公司 logo、頁碼位置)
- 建議使用自己統一設計的母片 (master slide)

### 2. 標題和本文的字型種類和大小
- 標題字型統一(種類、大小、粗細、顏色)
- 本文字型統一
- 層級關係清楚且全篇一致

### 3. 顏色和顏色數量
- 限制主色數量(建議主色 + 輔色 + 強調色,共 2–3 色)
- 從頭到尾使用企業色彩 (corporate color) 可增強品牌感
- 避免每頁用不同配色

### 4. 項目清單上的符號使用
- bullet 符號統一(●、✓、▸ 擇一)
- 編號格式統一(1. 2. 3. 或 ① ② ③)
- icon 風格一致(線條 / 填色 / 粗細)

## 補充:其他需留意的一致性元素
- 動畫效果 (animation)
- 投影片畫面切換 (transition)
- 圖表配色與樣式
- 圖片濾鏡與邊框

## 關鍵原則
> 使用自己統一設計的投影片,是保持一致性的最佳方法。人類的眼睛對細微差異的感知非常敏銳——細節的不一致會立刻被觀眾察覺。

## 應用 Checklist (可用於 prompt / skill)
製作或審查投影片時,逐項確認:
- [ ] 版型是否統一?
- [ ] 標題字型、大小、顏色是否一致?
- [ ] 本文字型、大小、顏色是否一致?
- [ ] 全篇使用色彩是否控制在 3 色以內?
- [ ] 是否從頭到尾使用一致的主色系?
- [ ] bullet 符號是否統一?
- [ ] 編號格式是否統一?
- [ ] icon 風格是否一致?
- [ ] 動畫與切換效果是否統一?
- [ ] 每頁是否都有配置一致元素(logo、頁碼)?
