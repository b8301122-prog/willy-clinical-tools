# 投影片技巧 #07：用明確的主詞消除不安

> 書名：你的簡報就是比AI強：47個人類才懂的PPT技巧
> 章節:1-7

## 核心原則
簡報中**主詞的有無與選擇**會大幅改變觀眾對訊息的印象與理解。省略主詞雖能讓文字更精簡,但若連必要的主詞都省略,會導致語意模糊、觀眾感到不安(「我這樣理解對嗎?」),反而本末倒置。

## 為什麼主詞重要(認知心理學依據)
和主詞不明確的情況相比,明確的主詞能產生五項正面效果:
1. 減輕認知負擔
2. 提高資訊的一致性及明確性
3. 提高記憶的扎根
4. 防止誤會
5. 提升興趣

主詞曖昧 → 因果關係不明 → 觀眾混亂或不安
主詞明確 → 「何人」「何事」清楚 → 訊息正確傳達

## 主詞的三種策略性運用

| 主詞 | 使用時機 | 效果 |
|------|---------|------|
| **我** | 強調個人的責任或參與時 | 表現個人意志與欲求,觀眾關注提案者的動機 |
| **自家公司／公司** | 強調組織利益或效率時 | 基於組織立場,觀眾以更遼闊的視野解讀 |
| **無(省略)** | 強調客觀事實或邏輯性觀點時 | 觀眾從客觀角度解讀主張 |

### 範例對照(以「導入敏捷軟體開發」提案為例)
- ❶「**我**想導入敏捷軟體開發」→ 個人意志,焦點在提案者動機
- ❷「**我們公司**應該導入敏捷軟體開發」→ 組織立場,焦點在公司整體利益
- ❸「應該導入敏捷軟體開發」→ 客觀主張,焦點在事實與邏輯

## BEFORE / AFTER 範例
**BEFORE(主詞不明)**
> 「邊做事邊滑手機」越來越受歡迎
> 近年來更成為十分熱門的資訊蒐集方式

問題:沒有主詞,不知道對象是誰;第二句似乎與第一句連結,但整體模糊。

**AFTER(主詞明確 + 具體例)**
> **Z世代**越來越喜歡「邊做事邊滑手機」。
> 不光是 SNS,廣播也是非常受歡迎的資訊源

改善:明確指出主詞是 Z 世代,並用「廣播」當作具體範例,資訊更清晰。

## 應用檢查清單(可放入 prompt / skill)
製作或檢查投影片文案時,逐項自問:
- [ ] 每個關鍵句子的**主詞是否明確**?若省略,是否為了刻意營造客觀感?
- [ ] 主詞選擇是否與**訊息意圖**一致(個人責任 / 組織立場 / 客觀事實)?
- [ ] 是否存在主詞曖昧導致的**因果關係不明**?
- [ ] 抽象敘述是否搭配**具體範例**,讓主詞所指對象更鮮明?
- [ ] 觀眾讀完後,會不會產生「這是在說誰?」的不安?

## 給 Claude 的指令模板
> 檢查以下投影片文案的主詞使用:
> 1. 標示每個句子的主詞(明示／省略)
> 2. 判斷省略是否合理(客觀陳述 OK,關鍵主張不 OK)
> 3. 對於主詞曖昧的句子,依據訊息意圖(個人／組織／客觀)建議三種改寫版本
> 4. 指出是否需要補充具體範例來強化主詞所指對象
