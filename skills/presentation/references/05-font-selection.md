# 投影片字體選用原則

> **書名**:你的簡報就是比AI強:47個人類才懂的PPT技巧
> **章節**:1-5 利用哥德體及明體來調整簡報的易讀性

## 核心原則
簡報以**哥德體(無襯線體)為主**,明體(襯線體)作為強調或區隔用途點綴使用。

## 為什麼簡報要用哥德體?
- 哥德體筆畫粗細均一,**距離螢幕較遠仍可清楚辨識**
- 明體線條較細,在液晶螢幕上可能因解析度而辨識度下降
- 哥德體給人「隨意、友善」的印象,閱讀負擔較低

## 明體的有效使用方法

### 設計意圖
**用字體差異來「視覺分層」**——當一張投影片同時有「自己的論述」和「引用內容」時,若全部用同一種哥德體,讀者無法一眼分辨哪些是引用、哪些是作者觀點。

### 實作手法(Before → After 案例)
**情境**:投影片引用名人名言來支持論點

- **❌ Before**:標題、論述、引用全部用哥德體 → 引用文字混在內文中,讀者看不出哪一段是名言
- **✅ After**:
  - 標題、論述 → 哥德體
  - **引用段落 → 改用明體,並加上淺灰底色框**
  - 結果:引用部分自然「凸出」,層次分明

### 適用時機
- **名言、引述、出處段落**(最經典用法)
- **強調重點**:與其他內容做出區隔
- **議程、簡報摘要、商品資訊標示**
- ⚠️ 過度使用會造成閱讀困難與視覺混亂——明體應該是「點綴」,不是主體

## 推薦字體(依平台)

| 平台 | 首選哥德體 | 備註 |
|---|---|---|
| **Mac** | HiraginoKaku Gothic | 字距較寬、縮小仍清楚、適讀性最佳;可搭配 HiraginoMincho |
| **Windows** | Yu Gothic Medium | PowerPoint 預設無 Hiragino,改用游哥德體 |
| **柔和主題** | Maru Gothic(圓哥德體) | 圓潤字型降低觀眾抗拒感,適合艱澀主題 |

## 四種應該認識的字體

### 哥德體(簡報主力)
- **HiraginoKaku Gothic**(Mac):辨識度高、比例勻稱、漢字/平假名/片假名和諧搭配,多種字重可選。⚠️ Windows/Mac 間有相容性問題
- **Yu Gothic**(Windows PowerPoint 預設):精緻、適合螢幕與印刷。線條較纖細,**標題建議用粗體或 Medium**

### 明體(強調/引用用)
- **MS Mincho**:較粗、易讀,比例佳。⚠️ 小字易變形
- **Yu Mincho Light**:輕盈、現代感,適合**大尺寸標題、海報**製造視覺衝擊。⚠️ 低解析度環境不易閱讀

## 應用 Checklist
- [ ] 內文是否用哥德體?
- [ ] 投影片是否有「引用/名言」段落?若有,是否改用明體 + 底色框做出視覺區隔?
- [ ] 明體是否只用在「需要區隔/強調」之處,沒有過量?
- [ ] 標題若用 Yu Gothic,是否提升到 Medium 或 Bold?
- [ ] 跨平台分享時是否考慮字體相容性?
- [ ] 小字部分是否避開纖細明體?

## 一句話口訣
> **哥德體打底,明體點睛;引用名言時,換字體 + 加底色,讓層次自己說話。**
