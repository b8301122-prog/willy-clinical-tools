# 簡報的顏色請控制在 3 色以內

> **書名**：你的簡報就是比 AI 強：47 個人類才懂的 PPT 技巧
> **章節**：2-6

---

## 核心原則

- **1 張投影片最多 3 種顏色**：超過 4 色會失去設計一致感,重點強調效果減半
- **重點色使用比例 ≤ 10%**:只在最希望吸引觀眾注意的部分使用強調色
- **顏色是資訊的一部分**:跟文字一樣要有邏輯地運用,不能憑感覺安排

## 顏色的心理效果

| 顏色 | 聯想／情緒 |
|---|---|
| 紅色 | 危險、熱情、活力、重要性 |
| 黃色 | 開朗、快樂 |
| 綠色 | 自然、富饒、環保（有益環境） |
| 藍色 | 沉靜、信賴、環保或潔淨（有益環境） |
| 紫色 | 高貴、神祕、非常危險 |

> 選色要配合訊息語境:例如強調「環保產品」時用綠色,比用紅色更能喚起正確聯想。紅色雖吸引注意,但不一定適用於正面訊息。

## 對比色運用

當使用 2 色以上時,應優先考慮「對比色」:

- **最基本的兩組對比色**:
  - 🔴 紅 × 🟢 綠
  - 🟡 黃 × 🔵 藍
- **原理**:人類無法同時感受紅綠或藍黃兩種色調,因此對比最強烈
- **色相環上正對位置 = 對比關係**
- **亮度差異會影響用途**:
  - 紅綠:亮度相近,難以評論何者沉重
  - 黃藍:黃色較明亮,藍色感覺更沉重

## 實作規則（BEFORE → AFTER 歸納）

### Rule 1：只在重點處上色
- ❌ BEFORE:白底 + 紅 + 綠 + 黃三色並列標題,焦點模糊
- ✅ AFTER:主體用黑字,只對比色（紅/綠）標示希望引起注意的重點
- **判準**:如果一張投影片每個元素都被上色,等於沒有重點

### Rule 2：顏色要呼應訊息語意
- 介紹「環保產品」→ 用綠色（聯想環境）
- 警告／危險／關鍵數字 → 紅色
- **不要因為好看而選色**

### Rule 3：矩陣圖／分類圖不需要用顏色區分每一類
- ❌ BEFORE:5 家公司用 5 種顏色標示 → 雜亂、無重點
- ✅ AFTER:只幫「想強調的那一家」上色,其餘用灰階
- **原則**:用顏色抑制而非增加視覺負擔

### Rule 4：座標軸強調要有目的
- 不要在縱軸兩端都用黃色（無意義）
- 只在「優勢方向」的軸標籤上色,表達方向性

## Prompt／Skill 可複用片段

```markdown
## 簡報配色規範（Color Discipline）

1. **3 色上限原則**:單張投影片顏色總數 ≤ 3（含強調色）,超過視為違規。
2. **10% 重點色規則**:強調色面積佔比不超過整張投影片的 10%。
3. **語意配色**:
   - 紅色:警示、危險、關鍵重點、負面數據
   - 綠色:環保、自然、正面成長、安全
   - 藍色:信賴、沉穩、資訊類主色
   - 黃色:明亮提醒(慎用於白底,對比度低)
   - 紫色:高貴或極端警告
4. **對比色優先**:需要兩色並陳時,使用 紅綠 或 黃藍 對比組合。
5. **抑制優於點綴**:分類圖中僅對「想強調的對象」上色,其餘以灰階處理。
6. **禁止事項**:
   - 白底 + 黃字(對比度不足)
   - 每個項目都換不同顏色
   - 為了「好看」而上色(顏色必須承載語意)
```
