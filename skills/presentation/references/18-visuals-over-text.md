# 投影片製作技巧 #01：善用圖表、插畫、照片取代文字

> **書名**：你的簡報就是比 AI 強：47 個人類才懂的 PPT 技巧
> **章節**：4-1 比起文字，圖表、插畫、照片更好理解

## 核心原則
視覺資訊比純文字更容易被大腦處理與記憶。製作投影片時，應優先以圖像承載資訊，文字僅作為輔助。

## 關鍵研究與數據

- **記憶留存率（72 小時後）**
  - 純文字:約 10%
  - 文字 + 影像：約 65%
  - 來源：John Medina《Brain Rules（大腦當家）》

- **雙碼理論（Dual Coding Theory）**
  - 提出者：Allan Paivio(認知心理學家)
  - 人類記憶由兩個系統構成：處理語言的「語言系統」與處理視覺的「非語言系統」
  - 結論：用圖像記憶名詞的準確率，是用文字記憶的兩倍

- **單張投影片理想文字量**：約 120 字（上限，非目標）

## 為什麼圖像有效
1. 視覺資訊透過「本能層次」處理，不需經過邏輯推演，接收速度快
2. 一張插畫可同時傳達多個訊息（場景、人物、動作、關係），若改用文字需大量篇幅
3. 長篇文字會降低觀眾閱讀意願,資訊傳遞效率反而下降

## 實作準則

| 原則 | 具體做法 |
|------|---------|
| 化繁為簡 | 處理複雜事項或因果關係時，優先用插畫/照片 |
| 資訊量最佳化 | 圖像提供更多資訊，同時把字數壓到最低 |
| 剔除多餘 | 任何能用圖表達的，就不要用文字 |
| 適用情境 | 因果關係、流程、場景、空間關係、比例、趨勢 |

## Before / After 範例
- **Before**：一整段描述「數學課上老師向 3 位孩子解說黑板圖形，左邊男孩舉手發問，右邊男孩似乎不太理解」
- **After**：一張教室插畫，觀眾一眼就能讀出「上課中」「3 個小孩」「左邊舉手」，甚至細節如「時間 2:50」

## 應用 Checklist（給 Prompt / Skill 用）
製作或審閱投影片時，逐項檢查：
- [ ] 這張投影片的文字是否 ≤ 120 字？
- [ ] 有沒有任何資訊可以用圖表、插畫、照片取代？
- [ ] 複雜的因果/流程/關係，是否已視覺化？
- [ ] 圖像是否承擔主要訊息，文字僅為輔助標註？
- [ ] 是否已剔除所有不必要的文字與裝飾？
