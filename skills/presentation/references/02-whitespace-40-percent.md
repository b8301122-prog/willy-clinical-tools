# 投影片製作技巧 #02：適時留白，讓人「意猶未盡」

## 核心原則

**留白應佔整張投影片約 40%**,用以維持簡潔印象、降低視覺壓力與認知負荷。

---

## 為什麼需要留白?

### 避免「視覺壓力」(Visual Stress)
- 視覺接收到過量資訊時,會造成身體與精神上的壓力
- 極端例子:密集恐懼症 (Trypophobia)
- 塞滿文字的投影片即使未達恐懼程度,仍會讓觀眾感到壓迫感

### 減輕認知負荷
- 資料中的文字等元素越少 → 越能減緩視覺壓力
- 對觀眾而言更易閱讀
- 創造「意猶未盡」的感覺,比塞滿內容更恰當

---

## 留白的具體配置

| 位置 | 比例 |
|------|------|
| 左右各留 | 約 10% |
| 上下各留 | 約 10% |
| **合計** | **約 40%** |

### 補救方案
若四邊無法各留一成,可透過以下方式補足:
- **標題與內文之間的空白**
- **字距**(字與字間距)
- **行距**(行與行間距)

---

## BEFORE vs AFTER 對照

### ❌ BEFORE(難閱讀)
- 文字量大,邊緣兩側擁擠
- 幾乎沒有留白
- 為了塞入更多資訊而縮小字體
- 產生視覺壓力

### ✅ AFTER(易閱讀)
- 文字數量精簡
- 上下左右都有足夠留白
- 字體尺寸較大
- 簡潔、直覺、易讀

---

## Prompt / Skill 應用版本(可直接貼用)

```markdown
## 投影片留白原則

設計投影片時請遵守以下留白規範:

1. **40% 留白規則**:整張投影片應保留約 40% 的留白面積
2. **邊距配置**:左右各留約 10%、上下各留約 10%
3. **文字精簡**:寧可減少文字量也不要縮小字體
4. **字體放大**:文字尺寸應足夠大,不應為塞入內容而縮小
5. **補救機制**:若邊距不足 40%,透過加大「標題與內文間距」、
   「行距」、「字距」來補足留白
6. **核心目標**:創造「意猶未盡」的感覺,避免「視覺壓力 (Visual Stress)」
   與認知負荷

避免:塞滿文字、縮小字體、邊緣擁擠、元素過密
追求:簡潔、直覺、易讀、足夠呼吸空間
```
